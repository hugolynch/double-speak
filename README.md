# Waterfalls (game)

Single-view Svelte 5 + Vite app for the daily word puzzle.

## Dev

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

- Production base: `/waterfalls/` (see `vite.config.ts`).
- Types live under `src/types`. State uses Svelte runes in `src/lib/state.svelte.ts`.
